# Contributing to Quarks

Thanks for your interest! This project needs contributors like you.

## Quick Start

Clone/download repo, then install dependencies (libportaudio, inputlib, sounddevice, numpy e.t.c)

So you can launch the quark-sounds.py directly from source code.

## How to Contribute

### Issues

#### Idea to introduce something new?
- Open an issue, that clearly explains what you would like to see in the future update.

#### Found a bug?
- Open an issue, with instructions for reproducing the bug.

### Pull Requests

#### If you want to contribute your changes directly to the project, folow instruntions below

1. Fork the repo ("Fork" button on the top right)
2. Create a new branch for your forked repo, naming it with readable/understandable name
3. Commit all your changes, optionally explaining changes
4. Go to https://github.com/quadakr/quarks/pulls, and click "New pull request"
5. Select your branch, describe what you changed, and submit request

If anything is unclear, you can open an issue and ask questions
