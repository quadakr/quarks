# quarks
Simple cli tool for white noise real-time generation depended on uder actions.

# What's "cool"?
Using libinput, this script smoothly increase noise frequency when typing speed increases. (or simply, when you fastly press keyboard buttons)

# Early Developement
Code is not clear yet, behaviour may change during project enhancement.

# How to set up
Install python3, libinput, numpy (python library), sounddevice (python library too), and it would work on your machine.
